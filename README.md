# Dice Throne Adventures Portal Crawl Randomizer

A web-based randomizer for **Dice Throne Adventures** that generates complete Portal Crawl setups without requiring the physical Portal Crawl cards.

Whether you've misplaced your cards, want unlimited randomized layouts, or want to share the exact same adventure with friends, this tool has you covered.

## Features

- 🎲 Generate Portal Crawls for:
  - Tier 1
  - Tier 3
  - Tier 5
  - Tier 7
- 🌱 Seed support
  - Generate a random seed
  - Copy and share seeds with friends
  - Paste a seed to recreate the exact same Portal Crawl
- 🦸 Includes Hero Setup information
- 👑 Displays King's Hand Token setup
- 🏁 Includes Conclusion information
- 📖 Built-in legend explaining map symbols and icons
- ⚡ Fast, lightweight, and easy to use

## Why?

The physical Portal Crawl cards included with Dice Throne Adventures offer a limited number of layouts. This randomizer expands those possibilities while remaining fully reproducible through shareable seeds.

## How to Use

1. Select the Portal Crawl tier you want to play.
2. Generate a new crawl, or enter an existing seed.
3. Share the generated seed with friends if you want friends to play the exact same adventure.
4. Set up your board using the generated Portal Crawl, Hero Setup, King's Hand Tokens, and Conclusion information.
5. Enjoy your adventure!

## Seed System

Every generated Portal Crawl is associated with a unique seed.

- Generate a random seed for a new adventure.
- Copy the seed to share with friends.
- Paste the seed later to recreate the exact same Portal Crawl.

This makes it easy to replay favorite adventures or challenge other groups with identical setups.
